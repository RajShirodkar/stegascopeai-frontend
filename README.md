# StegaScopeAI Frontend

StegaScopeAI is a web-based AI-powered steganalysis platform designed to detect
hidden information inside digital images.

This repository contains the **frontend** of the project, built using:

- Next.js (App Router)
- Tailwind CSS
- Modern responsive UI design

## Features

- Landing page
- User authentication UI (Sign In / Sign Up)
- Dashboard with image upload interface
- Clean, responsive, and user-friendly design
- Ready for backend & AI model integration

## Tech Stack

- Next.js
- React
- Tailwind CSS

```bash
npm install
npm run dev
